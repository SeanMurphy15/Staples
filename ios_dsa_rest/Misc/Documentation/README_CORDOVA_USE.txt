REST DSA + Cordova/PhoneGap project

Installation

Checkout this project to your Mac machine with Xcode 4.5 installed.


Build

Main project file - iOS_DSA_REST_Cordova.xcworkspace
Main target - ios_dsa

Get copy of PhoneGap/Cordova here - https://nodeload.github.com/phonegap/phonegap/zipball/2.1.0, unzip it and put into any appropriate folder.

Next instructions you can find here - http://docs.phonegap.com/en/2.1.0/guide_getting-started_ios_index.md.html#Getting%20Started%20with%20iOS
