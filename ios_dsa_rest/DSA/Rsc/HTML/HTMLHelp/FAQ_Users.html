How many categories can I have in the DSA?

Theoretically, the application supports unlimited categories and products. However, categories can only be two levels deep.

Does Single Sign On (SSO) work with DSA?

If your Salesforce instance is set up to accept your SSO credentials, the DSA app works with SSO and leverages OAuth2 authentication with those credentials.

What information is stored on the iPad?

Your contacts, product categories, and content (images, documents, and videos) are stored on the iPad and automatically updated whenever an Internet connection is detected.

What if an Internet connection is not available in the field?

Since the content from your Salesforce account is stored on the device, all features are available even when Internet connectivity is not.

What are the system requirements for running DSA?

DSA has the following requirements for the iPad and Salesforce:

iPad Salesforce

iPad 1 or 2
WiFi or 3G model
Adequate space on device to store content

 Salesforce license
App Exchange package
Administrative rights to install package
Configuration of DSA through Salesforce (completed by administrator)
Can I send any information to contacts from the application?

Images and documents (PowerPoint presentations, PDF files, Word documents, etc.,) can be forwarded to contacts via e-mail. Please note that a valid e-mail account must be set up on your iPad to use this feature.

What if I don't want to allow documents to be emailed?

Permission to e-mail documents is defined in Salesforce. The document type should be set to "competitive information" in the Salesforce Content settings.
